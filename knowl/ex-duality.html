Exercise 5.

Consider Corollary 4.2.6.
(a)
Prove that if the primal program is infeasible, and the dual program is feasible, then the dual program is unbounded below. (Use Corollary 4.2.6.)
(b)
Prove that if the dual program is infeasible, and the primal program is feasible, then the primal program is unbounded above. (Use (a).)
(c)
Prove that if the primal (dual) program is feasible and unbounded above (below), then the dual (primal) program is infeasible.
(d)
Find a primal-dual linear program where both the primal and dual problems are infeasible.
