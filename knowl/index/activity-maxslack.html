Activity 2.1.1.

Consider the canonical maximization problem:
\begin{align*} \textbf{Maximize: } f(\mathbf{x}) = c_1x_1 + c_2x_2 +\cdots c_mx_m-d \amp= \left(\displaystyle\sum_{j=1}^m c_jx_j \right)-d\\ \textbf{subject to: } a_{1,1}x_1 + a_{1,2}x_2 +\cdots a_{1,m}x_m\amp \leq b_1\\ a_{2,1}x_1 + a_{2,2}x_2 +\cdots a_{2,m}x_m\amp \leq b_2\\ \vdots \amp \vdots\\ a_{n,1}x_1 + a_{n,2}x_2 +\cdots a_{n,m}x_m\amp \leq b_n\\ x_1, x_2, \ldots, x_m\amp \geq 0. \end{align*}
Note that we may rewrite this as a system of equalities by introducing the slack variables \(t_i\text{.}\) We will refer to the \(x_j\) as decision variables.
\begin{align*} \textbf{Maximize: } f(\mathbf{x}) = c_1x_1 + c_2x_2 +\cdots c_mx_m-d \amp= \left(\displaystyle\sum_{j=1}^m c_jx_j \right)-d\\ \textbf{subject to: } a_{1,1}x_1 + a_{1,2}x_2 +\cdots a_{1,m}x_m + t_1 \amp = b_1\\ a_{2,1}x_1 + a_{2,2}x_2 +\cdots a_{2,m}x_m+t_2\amp = b_2\\ \vdots \amp \vdots\\ a_{n,1}x_1 + a_{n,2}x_2 +\cdots a_{n,m}x_m +t_n \amp = b_n\\ x_1, x_2, \ldots, x_m\amp \geq 0. \end{align*}

(a)

What must be true about the slack variables \(t_i\text{?}\)
  1. \(\displaystyle t_i\leq 0.\)
  2. \(\displaystyle t_i\geq 0.\)
  3. \(\displaystyle \sum_{i=1}^m t_i=0.\)

(b)

Which of these must be an upper bound for \(t_i\) ?
  1. \(\displaystyle b_i.\)
  2. \(\displaystyle c_i.\)
  3. \(\displaystyle x_i.\)
  4. \(t_i\) could be unbounded.
