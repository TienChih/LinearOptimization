Activity 4.3.1.

Noting that the dual variables \(y_i\) are non-negative weights attached to the hyperplanes defined by \(t_i=0\text{,}\) and the slack variables for the dual program \(s_j\) are the weights associated with the planes \(x-x_j=0\text{,}\) we can encode all this information in the Primal-Dual Tucker Tableau:
\(x_1\) \(\cdots\) \(x_m\) \(-1\)
\(y_1\) \(a_{11}\) \(\cdots\) \(a_{1m}\) \(b_1\) \(=-t_1\)
\(\vdots\) \(\vdots\) \(\ddots\) \(\vdots\) \(\vdots\) \(\vdots\)
\(y_n\) \(a_{n1}\) \(\cdots\) \(a_{nm}\) \(b_n\) \(=-t_n\)
\(-1\) \(c_1\) \(\cdots\) \(c_m\) \(d\) \(=f\)
\(=s_1\) \(\cdots\) \(=s_m\) \(=g\)
(The additional dividing lines in the top left and bottom right seperate the primal decision-slack variables from the dual decision-slack vasriables.)

(a)

Write out the sufficient conditions for the tableau to determine:
  1. The primal program is feasible.
  2. The dual program is feasible.
  3. A feasible primal program is unbounded above.
  4. A feasible dual program is unbounded below.
  5. A primal program is infeasible.
  6. A dual program is infeasible.
  7. The primal program has a feasible basic solution.
  8. The dual program has a feasible basic solution.

(b)

Are any of these identical?
