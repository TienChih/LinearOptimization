Activity 7.3.3.

In this activity, we model the shortest path algorithm as a linear programming problem. Assume \(N\) is a weighted network with no negative cycles. Let \(0\leq x_{ij}\leq 1\) where \(x_{ij}=1\) if \((v_i, v_j)\) is in a shortest path \(P\) from \(v_s\) to \(v_d\text{.}\)

(a)

What is the objective problem?
  1. Maximize \(\displaystyle \sum_i w_{it}x_{it} \text{.}\)
  2. Maximize \(\displaystyle \sum_j w_{sj}x_{sj} \text{.}\)
  3. Maximize \(\displaystyle \sum_{i,j} w_{ij}x_{ij} \text{.}\)
  4. Minimize \(\displaystyle \sum_i w_{it}x_{it} \text{.}\)
  5. Minimize \(\displaystyle \sum_j w_{sj}x_{sj} \text{.}\)
  6. Minimize \(\displaystyle \sum_{i,j} w_{ij}x_{ij} \text{.}\)

(b)

What inequality ensures that exactly one edge of the chosen edges is incident to \(v_d\text{?}\)
  1. \(\displaystyle \sum_i x_{id} \leq 1\text{.}\)
  2. \(\displaystyle \sum_i x_{id} \geq 1\text{.}\)
  3. \(\displaystyle \sum_i x_{id} = 1\text{.}\)
  4. \(\displaystyle \sum_i x_{id} \leq 0\text{.}\)
  5. \(\displaystyle \sum_i x_{id} \geq 0\text{.}\)
  6. \(\displaystyle \sum_i x_{id} = 0\text{.}\)

(c)

What inequality ensures that the chosen edges form a path?
  1. For each vertex \(v_i\neq v_s, v_d\text{,}\) \(\displaystyle \sum_{j} x_{ji} - \sum_{j} x_{ij}=1\text{.}\)
  2. For each vertex \(v_i\neq v_s, v_d\text{,}\) \(\displaystyle \sum_{j} x_{ji} - \sum_{j} x_{ij}\leq 1\text{.}\)
  3. For each vertex \(v_i\neq v_s, v_d\text{,}\) \(\displaystyle \sum_{j} x_{ji} - \sum_{j} x_{ij}\geq 1\text{.}\)
  4. For each vertex \(v_i\neq v_s, v_d\text{,}\) \(\displaystyle \sum_{j} x_{ji} - \sum_{j} x_{ij}=0\text{.}\)
  5. For each vertex \(v_i\neq v_s, v_d\text{,}\) \(\displaystyle \sum_{j} x_{ji} - \sum_{j} x_{ij}\leq 0\text{.}\)
  6. For each vertex \(v_i\neq v_s, v_d\text{,}\) \(\displaystyle \sum_{j} x_{ji} - \sum_{j} x_{ij}\geq 0\text{.}\)
