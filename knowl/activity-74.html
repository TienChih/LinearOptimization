Activity 5.2.1.

Let \(A\) be a payoff matrix and \(\p, \q\) represent the strategies of the row and column players respectively, with feasible regions \(F_\p, F_\q\text{.}\)
Let \(A_{(i)}\) denote the \(i\)th row of a matrix \(A\) and let \(A^{(j)}\) denote the \(j\)th column of \(A\text{.}\)

(a)

Given a fixed column strategy \(\q'\) which of these describes the role for the row player?
  1. \(\displaystyle \displaystyle \min_{\p \in F_{\p}} \p^\top A\q'\)
  2. \(\displaystyle \displaystyle \max_{\p \in F_{\p}} \p^\top A\q'\)
  3. \(\displaystyle \displaystyle \min_{\p \in F_{\p}} \sum_{i=1}^n \p_i A_{(i)}\q'\)
  4. \(\displaystyle \displaystyle \max_{\p \in F_{\p}} \sum_{i=1}^n \p_i A_{(i)}\q'\)

(b)

Given a fixed row strategy \(\p'\) which of these describes the role for the column player?
  1. \(\displaystyle \displaystyle \min_{\q \in F_{\q}} \p'^\top A\q\)
  2. \(\displaystyle \displaystyle \max_{\q \in F_{\q}} \p'^\top A\q\)
  3. \(\displaystyle \displaystyle \min_{\p \in F_{\p}} \sum_{j=1}^m \p^\top A^{(j)}\q_j\)
  4. \(\displaystyle \displaystyle \max_{\p \in F_{\p}} \sum_{ij=1}^m \p^\top A^{(j)}\q_j\)
