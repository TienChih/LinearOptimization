Task 8.1.2.b.

Consider an integer programming maximization problem whose relaxation achieves an optimal solution. Which of the following are true about the integer program?
  1. The integer program achieves an optimal solution that is equal to the optimal solution of the relaxation.
  2. The integer program achieves an optimal solution that is less than to the optimal solution of the relaxation.
  3. The integer program achieves an optimal solution that is greater than to the optimal solution of the relaxation.
  4. The integer program is infeasible.
  5. The integer program is unbounded above.
