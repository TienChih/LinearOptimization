Task 2.3.2.f.

Find a sufficient condition for a feasible tableau for a canonical maximization problem to have an unbounded feasible region.
  1. All of the \(a_{ij}\leq 0\text{.}\)
  2. There is a row \(j\) so that all of the \(a_{ij}\leq 0\) in that row.
  3. There is a column \(i\) so that all of the \(a_{ij}\leq 0\) in that column.
  4. All of the \(c_j\leq 0\text{.}\)
  5. All of the \(b_i\leq 0\text{.}\)
