Definition 3.3.1.

  1. Ensure that each \(b_i\geq 0\text{,}\) that is that the solution is feasible.
  2. IF every \(c_j\leq 0\text{:}\) STOP the current solution is optimal.
  3. PICK a column \(j\) such that \(c_j > 0\text{.}\)
  4. IF each \(a_{1j}, a_{2j}, \ldots, a_{mj}\leq 0 \text{:}\) STOP the objective function is unbounded.
  5. Find a row \(i\) that minimizes \(\left\{ b_i/a_{ij}: b_i > 0 \right\}\text{.}\)
  6. PIVOT on \(a_{ij}\text{.}\)
  7. GOTO 1.
