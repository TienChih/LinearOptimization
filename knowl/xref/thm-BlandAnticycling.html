Theorem 3.5.3. Bland’s Anticycling Rules.

List all variables, independent and dependent in the initial tableau. Then consider the following rules:
  1. Whenever there is more than one possible choice for a pivot row according to Definition 3.3.1: select the row corresponding to the variable closest to the front of the initial list.
  2. Whenever there is more than one possible choice for a pivot column according to Definition 3.3.1: select the column corresponding to the variable closest to the front of the initial list.
Then the Simplex Algorthm will not cycle.
