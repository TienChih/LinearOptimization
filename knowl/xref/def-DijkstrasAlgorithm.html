Definition 7.3.4. Dijkstra’s Algorithm.

Let \(N\) be a weighted network with only non-negative weights. Then Dijkstra’s Shortest Path Algorithm is as follows:
  1. INITIALIZE: Let \(R=\{v_s\}\) and let \(T=V\backslash R\text{.}\) Label \(\ell_s=0\text{,}\) \(\ell_i = w_{si}\) if \(w_{si}\) exists, \(\infty\) otherwise.
  2. Let \(\ell_k=\displaystyle \min_{i\in T} \ell_i\text{.}\)
  3. Let \(R=R\cup \{v_k\}, T=V\backslash R\text{.}\)
  4. If \(T=\emptyset\text{:}\) STOP.
  5. For each \(v_j\in T\text{,}\) let \(\ell_j=\min\{\ell_j, \ell_j+w_{kj}\}\text{.}\)
  6. GOTO 2.
When the algorithm terminates, \(\ell_j = d(v_s, v_j)\text{,}\) the length of the shortest path from \(v_s\) to \(v_j\text{.}\)
