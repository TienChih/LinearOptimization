Theorem 1.3.22.

If \(S\) is the feasible region of a canonical problem and is unbounded:
  1. If the problem is a maximization problem and if there is a \(M\) so that \(f(s)\leq M\) for all \(s\in S\text{,}\) then \(S\) contains an optimal solution which is an extreme point of \(S\text{.}\)
  2. If the problem is a minimization problem and if there is a \(M\) so that \(g(s)\geq M\) for all \(s\in S\text{,}\) then \(S\) contains an optimal solution which is an extreme point of \(S\text{.}\)
