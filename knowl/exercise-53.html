Exercise 9.

Consider the payoff matrix
\begin{equation*} \begin{bmatrix} -x & x+1 \\ x+1 & -x-2\end{bmatrix} \end{equation*}
where \(x > 0\text{.}\)
(a)
Find the Von Nuemann value and optimal strategies for each player for the game above.
(b)
Suppose \(x\) could be any value, when does this matrix reduce via domination?
(c)
Find the Von Nuemann value and optimal strategies for each player for the game above if \(x\) could be any value.
