Suppose we wanted to solve the linear program:
\begin{align*} \textbf{Maximize: } f(\mathbf{x})= 3x+2y \amp \\ \textbf{Subject to: } 4x+5y\amp \leq 23\\ 2x +y\amp \leq 7\\ x -y\amp \leq 5 \end{align*}

(a)

What are some differences between this linear program and previous examples of linear programs?

(b)

We can record the program with the following tableau, we denote the variables which can be negative by circling them:
\(\ec{x}\) \(\ec{y}\) \(-1\)
\(4\) \(5 \) \(23\) \(=-t_1\)
\(2 \) \(1\) \(7 \) \(=-t_2\)
\(1\) \(-1 \) \(5 \) \(=-t_3\)
\(3 \) \(2 \) \(0\) \(=f\)
What point are we currently on?
