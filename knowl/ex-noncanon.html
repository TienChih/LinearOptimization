Exercise 1.

Solve each of the noncanonicallinear programming problems below. If a linear programming problem has infinitely many optimal solutions, find all optimal solutions.
(a)
\begin{align*} \textbf{Maximize: } f(x,y,z) \amp = x-2y-z\\ \textbf{Subject to: } x+y\amp \leq 1\\ x+y+z\amp \geq 2\\ y-2z \amp \geq 0 \end{align*}
(b)
\begin{align*} \textbf{Maximize: } g(x,y,z) \amp = 4x+2y+z\\ \textbf{Subject to: } x+y+z\amp \leq 5\\ 4x-y-z\amp \leq 5\\ 2y+z \amp \leq 4 \end{align*}
(c)
\begin{align*} \textbf{Maximize: } f(x,y,z) \amp = 4x+2y+z\\ \textbf{Subject to: } x+z\amp \leq 5\\ 4x-y-z\amp \leq 5\\ 2y+z \amp \leq 4 \end{align*}
(d)
\begin{align*} \textbf{Maximize: } k(x,y,z) \amp = x+2y+z\\ \textbf{Subject to: } 2x-y+z\amp = 3\\ x+2y\amp \leq 4\\ -2x+5y+3z \amp \leq 4\\ x, y, z \amp \geq 0 \end{align*}
(e)
\begin{align*} \textbf{Minimize: } h(x,y,z) \amp = x+y+z\\ \textbf{Subject to: } x-y\amp \geq 1\\ 2x+z\amp \leq 10\\ y+z \amp = 4\\ x, y, z \amp \geq 0 \end{align*}
(f)
\begin{align*} \textbf{Minimize: } f(x,y,z) \amp = 3x+y+2z\\ \textbf{Subject to: } x+2y+3z\amp \geq 24\\ 2x+4y+3z\amp = 36\\ y, z \amp \geq 0 \end{align*}
(g)
\begin{align*} \textbf{Maximize: } \ell(x,y,z) \amp = x+y+z\\ \textbf{Subject to: } x+y+z\amp \leq 3\\ x+y\amp \leq 1\\ y+2z\amp =2\\ x, y \amp \geq 0 \end{align*}
(h)
\begin{align*} \textbf{Maximize: } f(x,y,z) \amp = 3x-2y+3z\\ \textbf{Subject to: } x-y+2z\amp =6\\ x+2z\amp =8\\ y+2z\amp \geq 2\\ y, z \amp \geq 0 \end{align*}
(i)
\begin{align*} \textbf{Minimize: } \alpha(x,y,z) \amp = -5x+y-2z\\ \textbf{Subject to: } 2x+z\amp =0\\ x-y\amp \geq 1\\ 3x-y+z\amp \leq 3 \end{align*}
(j)
\begin{align*} \textbf{Maximize: } g(x,y,z) \amp = x+y+z\\ \textbf{Subject to: } x-y-z\amp \leq 2\\ y-z\amp \geq 1 \end{align*}
(k)
\begin{align*} \textbf{Maximize: } f(x,y) \amp = x+y\\ \textbf{Subject to: } 2x+y\amp =5\\ x-y\amp =-2\\ x+2y\amp =8\\ x, y \amp \geq 0 \end{align*}
