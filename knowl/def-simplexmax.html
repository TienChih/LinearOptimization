Definition 2.3.2.

The Simplex Algorithm for Maximum Tableaus
  1. IF each \(b_i\geq 0\text{,}\) GOTO 8.
  2. PICK the largest \(i\) such that \(b_i\lt 0\text{.}\)
  3. IF \(a_{i1}, a_{i2}, \ldots, a_{in}\geq 0 \text{:}\) STOP the problem in infeasible.
  4. PICK a column \(j\) such that \(a_{ij} \lt 0\) .
  5. PICK a row \(p\geq i\) so that \(\displaystyle \min_{k>i}\left(\{b_i/a_{ij}\} \cup \{b_k/a_{kj}:a_{kj}>0\} \right)=b_p/a_{pj}\text{.}\)
  6. PIVOT on \(a_{pj}\text{.}\)
  7. GOTO 1.
