Task 7.2.3.a.

Consider the dual program for this problem where \(\mu_{j}\) is the unconstrained dual variable for the vertex equality constraint and the \(y_{ij}\) is the dual variable for the capacity constraint. Verify that this program may be written as
\begin{align*} \textbf{Minimize: } \displaystyle \sum_{(v_i, v_j)\in E} c_{ij}y_{ij}\\ \textbf{Subject to: } \mu_j +y_{sj} \amp \geq 0, \text{for each edge }(v_s, v_j)\\ -\mu_i + \mu_j +y_{ij} \amp \geq 0, \text{for each edge} (v_i, v_j), i\neq s, j\neq d\\ - \mu_i +y_{id} \amp \geq 0, \text{for each edge }(v_i, v_d)\\ y_{ij}\amp \geq 0, \text{for each edge} (v_i, v_j) \end{align*}
where \(\mu_s=0, \mu_d=-1 \text{.}\)
