Definition 8.2.1. Gomory Cutting-Plane Algorithm.

The Gomory cutting-plane algorithm for an optimization problem is as follows:
  1. INITIALIZE: Begin with a canonical maximization integer programming problem.
  2. Solve the relaxation of the integer program. If all the resulting \(b_i\) are integral STOP: you have found an optimal integral solution.
  3. Select a \(b_i\) that is non integral and for that row, construct the additional bound: \(\sum_j (\lfloor a_{ij} \rfloor - a_{ij})x_j \leq (\lfloor b_i \rfloor - b_i)\text{.}\)
  4. GOTO 2.
