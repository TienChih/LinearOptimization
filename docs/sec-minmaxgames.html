    Definition 5.1.2.

    In a two-player zero-sum game, where the row player has \(n\) choices and the column player has \(m\) choices, the payoff matrix is a matrix which records in each row and column the net payoff to the row player (this choice is purely by convention, but we will stick to it).
    ğŸ”—
    If a row \(i\) has entries that are strictly greater than or equal to the entries of another row \(j\text{,}\) then we say that row \(i\) dominates row \(j\text{.}\) We then may delete row \(j\) since there is no reason the row player would choose \(j\text{.}\) Similarly, if column \(i\) is less than or equal to column \(j\text{,}\) column \(i\) dominates column \(j\) and we may delete column \(j\text{.}\)

    ğŸ”—
