    With all this, we finally may define our Simplex Algorithm for Maximum Basic Feasible Tableaus.

    Definition 3.3.1.

    The Simplex Algorithm for Maximum Basic Feasible Tableaus
    1. Ensure that each \(b_i\geq 0\text{,}\) that is that the solution is feasible.
    2. IF every \(c_j\leq 0\text{:}\) STOP the current solution is optimal.
    3. PICK a column \(j\) such that \(c_j > 0\text{.}\)
    4. IF each \(a_{1j}, a_{2j}, \ldots, a_{mj}\leq 0 \text{:}\) STOP the objective function is unbounded.
    5. PICK a row \(i\) that minimizes \(\left\{ b_i/a_{ij}: a_{ij} > 0 \right\}\text{.}\)
    6. PIVOT on \(a_{ij}\text{.}\)
    7. GOTO 1.

    With this, we may define our algorithm for potentiall non-feasible tableau

    Definition 3.3.2.

    The Simplex Algorithm for Maximum Tableaus
    1. IF each \(b_i\geq 0\text{,}\) GOTO 8.
    2. PICK the largest \(i\) such that \(b_i\lt 0\text{.}\)
    3. IF \(a_{i1}, a_{i2}, \ldots, a_{in}\geq 0 \text{:}\) STOP the problem in infeasible.
    4. PICK a column \(j\) such that \(a_{ij} \lt 0\) .
    5. PICK a row \(p\geq i\) so that \(\displaystyle \min_{k>i}\left(\{b_i/a_{ij}\} \cup \{b_k/a_{kj}:a_{kj}>0\} \right)=b_p/a_{pj}\text{.}\)
    6. PIVOT on \(a_{pj}\text{.}\)
    7. GOTO 1.
    8. APPLY Definition 3.3.1 .
