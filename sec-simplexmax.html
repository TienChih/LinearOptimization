    With all this, we finally may define our Simplex Algorithm for Maximum Basic Feasible Tableaus.

    Definition 2.2.1.

    The Simplex Algorithm for Maximum Basic Feasible Tableaus
    1. Ensure that each \(b_i\geq 0\text{,}\) that is that the solution is feasible.
    2. IF every \(c_j\leq 0\text{:}\) STOP the current solution is optimal.
    3. PICK a column \(j\) such that \(c_j > 0\text{.}\)
    4. IF each \(a_{1j}, a_{2j}, \ldots, a_{mj}\leq 0 \text{:}\) STOP the objective function is unbounded.
    5. PICK a row \(i\) that minimizes \(\left\{ b_i/a_{ij}: a_{ij} > 0 \right\}\text{.}\)
    6. PIVOT on \(a_{ij}\text{.}\)
    7. GOTO 1.

    Definition 2.2.2.

    The Simplex Algorithm for Maximum Tableaus
    1. IF each \(b_i\geq 0\text{,}\) GOTO 8.
    2. PICK the largest \(i\) such that \(b_i\lt 0\text{.}\)
    3. IF \(a_{i1}, a_{i2}, \ldots, a_{in}\geq 0 \text{:}\) STOP the problem in infeasible.
    4. PICK a column \(j\) such that \(a_{ij} \lt 0\) .
    5. PICK a row \(p\geq i\) so that \(\displaystyle \min_{k>i}\left(\{b_i/a_{ij}\} \cup \{b_k/a_{kj}:a_{kj}>0\} \right)=b_p/a_{pj}\text{.}\)
    6. PIVOT on \(a_{pj}\text{.}\)
    7. GOTO 1.
    8. APPLY Definition 2.2.1 .

    Subsection 2.2.3 The Simplex Algorithm for Canonical Minimization

    Definition 2.2.3.

    Recall that the canonical minimization problem:
    \begin{align*} \textbf{Minimize: } g(\mathbf{x}) = c_1x_1 + c_2x_2 +\cdots c_nx_n-d \amp= \left(\displaystyle\sum_{j=1}^n c_jx_j \right)-d\\ \textbf{Subject to: } a_{1,1}x_1 + a_{1,2}x_2 +\cdots a_{1,n}x_n\amp \geq b_1\\ a_{2,1}x_1 + a_{2,2}x_2 +\cdots a_{2,n}x_n\amp \geq b_2\\ \vdots \amp \vdots\\ a_{m,1}x_1 + a_{m,2}x_2 +\cdots a_{m,n}x_n\amp \geq b_m\\ x_1, x_2, \ldots, x_n\amp \geq 0 \end{align*}
    Where \(a_{i,j}, b_i, c_j, d\in \mathbb{R}\text{.}\)
    This problem may be recorded in the following tableau:
    \(x_1\) \(a_{11}\) \(a_{12}\) \(\cdots\) \(a_{1m}\) \(c_1\)
    \(x_2\) \(a_{21}\) \(a_{22}\) \(\cdots\) \(a_{2m}\) \(c_2\)
    \(\vdots\) \(\vdots\) \(\vdots\) \(\ddots\) \(\vdots\) \(\vdots\)
    \(x_n\) \(a_{n1}\) \(a_{n2}\) \(\cdots\) \(a_{nm}\) \(c_n\)
    \(-1\) \(b_1\) \(b_2\) \(\cdots\) \(b_m\) \(d\)
    \(=t_1\) \(=t_2\) \(\cdots\) \(=t_m\) \(=g\)
    The negative transpose of this tableau is:
    \(x_1\) \(x_2\) \(\cdots\) \(x_n\) \(-1\)
    \(-a_{11}\) \(-a_{21}\) \(\cdots\) \(-a_{n1}\) \(-b_1\) \(=-t_1\)
    \(-a_{12}\) \(-a_{22}\) \(\cdots\) \(-a_{n2}\) \(-b_2\) \(=-t_2\)
    \(\vdots\) \(\vdots\) \(\ddots\) \(\vdots\) \(\vdots\) \(\vdots\)
    \(-a_{1m}\) \(-a_{2m}\) \(\cdots\) \(-a_{nm}\) \(-b_n\) \(=-t_n\)
    \(-c_1\) \(-c_2\) \(\cdots\) \(-c_m\) \(-d\)

    Activity 2.2.10.

    Show that this negative transpose of a canonical minimization tableau encodes a problem that has the same feasible region and optimal solution as the original minimization problem.
    This allows us to define a minimization algorithm.

    Definition 2.2.4.

    The Simplex Algorithm for Minimization Tableaus
    1. TAKE the Negative Transpose.
    2. APPLY Definition 2.2.2 .
